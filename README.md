# Telegram-бот для поддержки

Бот с FAQ и поддержкой, плюс админ-меню для управления контентом.

## Возможности

- ❓ **FAQ** — список вопросов и ответов
- 💬 **Поддержка** — диалог с администраторами
- 🔧 **Админ-меню**:
  - управление FAQ (добавление/удаление)
  - управление администраторами
  - приглашение админов по реф-ссылке

## Установка

### 1. Создать бота

1. Откройте Telegram и напишите @BotFather
2. Выполните `/newbot`
3. Укажите имя бота (например: «Лебяжье озеро»)
4. Укажите username (например: `lebyajye_ozero_bot`)
5. Получите токен

### 2. Настроить конфиг

В `config.py` укажите `MAIN_ADMIN_ID`:

```python
MAIN_ADMIN_ID = 123456789  # ваш Telegram ID
```

Токен передаётся через локальную переменную окружения `API_TOKEN`.

PowerShell:

```powershell
$env:API_TOKEN="123456789:ABCdefGHIjklMNOpqrsTUVwxyz"
```

Узнать свой Telegram ID можно у @userinfobot.

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить бота

```bash
python main.py
```

## Команды

- `/start` — главное меню
- `/admin` — админ-меню (только для админов)
- `/help` — справка

## Как пригласить нового админа

1. Зайдите в админ-меню (`/admin`)
2. Нажмите «🔗 Пригласить админа (реф-ссылка)»
3. Отправьте полученную ссылку человеку
4. После перехода по ссылке пользователь станет админом

## Структура проекта

```
bot/
+-- main.py          # точка входа
+-- config.py        # конфигурация
+-- handlers.py      # обработчики сообщений и callback'ов
+-- keyboards.py     # клавиатуры
+-- database.py      # работа с SQLite
+-- data/            # данные и база
+   L-- bot.db       # SQLite база (создается автоматически)
+-- requirements.txt # зависимости
L-- README.md        # документация
```

## База данных

SQLite база `data/bot.db` содержит таблицы:

- **admins** — администраторы
- **faq** — вопросы/ответы
- **ref_tokens** — одноразовые реф-ссылки
- **active_chats** — активные чаты поддержки
